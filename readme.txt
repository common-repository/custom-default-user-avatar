Allow users to choose custom avatars for their user accounts.

== Description ==

Custom Default User Avatar is a fork of original [WP User Avatar](https://wordpress.org/plugins/wp-user-avatar/) plugin. Since its original functionality was removed after a significant overhaul by the plugin author, this plugin continues the original direction from v2.2.16. This version also includes security fixes and patches as suggested by WordPress Plugins Team.

== Installation ==

1. Download the plugin

2. Go to your Dashboard and locate Plugins tab. Click on Upload Plugin and specify the path of downloaded plugin in the text box.

3. Click on Activate Plugin
